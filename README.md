# Core_Member_Management
Better management


A lightweight activity planning platform for small friend groups.
Plan dinners, hikes, board game nights, and road trips in one place — with RSVP, time voting, location decisions, carpool assignments, and shared updates.

一个给小团体使用的活动管理平台。
从吃饭、hiking、桌游到 road trip，都可以集中管理时间、地点、参与人、车辆分配、集合点和更新记录，避免信息被群聊覆盖。
